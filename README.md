# 💰 Финансовая Грамотность - Образовательный сайт

Интерактивный сайт для обучения основам финансовой грамотности в стиле Duolingo.

## 🎯 Особенности

- **Интуитивный интерфейс** в стиле Duolingo с кружочками для уроков
- **6 тематических уроков** по финансовой грамотности
- **Теория + практика** для каждого урока
- **Интерактивные тесты** с мгновенной обратной связью
- **Отслеживание прогресса** обучения
- **Адаптивный дизайн** для всех устройств

## 📚 Содержание уроков

1. **Основы бюджетирования** - планирование доходов и расходов
2. **Банки и кредиты** - понимание банковских услуг
3. **Инвестиции** - основы инвестирования
4. **Кредитные карты** - ответственное использование
5. **Недвижимость** - покупка и аренда жилья
6. **Страхование** - защита финансовых рисков

## 🚀 Запуск проекта

1. Скачайте все файлы проекта
2. Откройте файл `index.html` в любом современном браузере
3. Начните обучение!

## 🎓 Как использовать

1. **Выберите урок** - кликните на кружочек с темой
2. **Изучите теорию** - прочитайте материал урока
3. **Пройдите тест** - ответьте на вопросы для закрепления
4. **Посмотрите результаты** - узнайте свой результат
5. **Продолжайте обучение** - переходите к следующему уроку
